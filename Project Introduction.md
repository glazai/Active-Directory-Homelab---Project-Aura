In this project we are building an Active Directory homelab that resembles the structure and policies of a real-world company.

The scenario
I have been hired as the IT Lead for AuraCorp, a growing startup. They currently have no central management.

The scope of this lab is:
- Domain Controller set up and configuration
- Organisational Unit structure
- Use and Group management
- Group policy objects
- File sharing and Permissions
- Security Hardening

Lab Prerequisites
- Windows Server 2022
- Virtual box
- Virtual Hardware: 2CPUs, 4GB RAM, 50GB Storage
